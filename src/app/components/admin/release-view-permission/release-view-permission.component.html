Quick Overview of Permissions:

Below are the permission types available in this screen for each configured release types / releases:

ADMIN: Only Admin users have access.

ADMIN, STAFF AND MEMBER: Access is granted to Admin, Staff and Member roles.

ADMIN, STAFF, MEMBER AND AFFILIATES: Access is granted to Admin, Staff, Member and all Affiliates.

ADMIN, STAFF, MEMBER AND SELECTED USERS: Access is granted to Admin, Staff, Member and specific individual users selected from a list.

EVERYONE: The release is public and accessible to all users.
