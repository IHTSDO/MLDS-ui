In this screen, there is a Release permission section check box to select the permission level for who can access the release.

Admin:
- This is the default selection.
- The Admin checkbox is always checked and cannot be unchecked.

Admin, Staff, Member:
- When these three checkboxes are selected, the release is visible only to these roles.

All Affiliates:
- Grants access to all affiliate users.
- Also checks Admin, Staff, and Member automatically.

Selected Users:
- Limits access to specific individual users.
- Admin, Staff, and Member are auto-selected.

Everyone:
- Makes the release public to all users.
